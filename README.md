# BrandSip Website

Your complete website for BrandSip — water bottle advertising.

## How to publish on GitHub Pages (free)

1. Go to **github.com** and create a free account
2. Click **"New repository"**
3. Name it exactly: `yourusername.github.io` (replace with your actual GitHub username)
4. Set it to **Public**
5. Click **"Create repository"**
6. Click **"uploading an existing file"**
7. Drag and drop `index.html` into the upload area
8. Click **"Commit changes"**
9. Go to **Settings → Pages**
10. Under "Source", select **"Deploy from a branch"** → **main** → **/ (root)**
11. Click Save — your site will be live at `https://yourusername.github.io` within 2-3 minutes! 🎉

## Before publishing — update these

Open `index.html` and find:

- `91XXXXXXXXXX` — replace with your actual WhatsApp number (no spaces or dashes)
- Example: if your number is 9845012345, change it to `919845012345`

## Later — adding photos to the gallery

When you have photos from your first distribution:
1. Upload your images to the repository
2. Replace the gallery placeholder section in index.html with `<img>` tags

## Adding a custom domain (optional, ~₹800/year)

Buy a domain from GoDaddy or Namecheap, then in GitHub Pages settings add your custom domain.
